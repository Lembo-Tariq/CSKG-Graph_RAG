My Role: RA at the University of Sharjah under Dr. Danilo Dessi as a mentor.
This project aim to use the CSKG 2.0 and graph RAG together to get better retrievial of infromation, hence more accurate results from the CSKG 2.0.
Achieving a much accurate answers and better understanding of the research topics.
Output of the project: Accurate efficient scientifc retrieval tool for the research community.

methodology: Start by Parsing a sample from the CSKG 2.0 into text, chunking and vectorzing, create a baseline RAG, and see results. Next Step is enhancing the baseline RAG.
